# 🎮 Tic Tac Toe - Java GUI with AI

A modern **Tic Tac Toe game** built using **Java Swing**, featuring a clean UI and a simple AI opponent.

---

## 🚀 Features

* 🧑 Player vs Computer gameplay
* 🤖 Random AI move logic
* 🎨 Modern dark-themed UI
* ✨ Smooth button styling & colors
* ⏱️ Delayed AI move for realism
* 🔄 Restart game anytime
* 🏆 Win detection + highlight
* 🤝 Draw detection

---

## 🛠️ Tech Stack

* **Java**
* **Swing (GUI)**
* **AWT Event Handling**

---

## 📂 Project Structure

```
Tic Tac Toe.java   # Main game file
```

---

## ▶️ How to Run

1. Compile the program:

```
 Tic Tac Toe.java
```

2. Run the program:

```
java Tic Tac Toe
```

---

## 🎯 How to Play

* Click any empty cell to place **X**
* The computer automatically plays **O**
* First to align 3 marks wins
* Game resets after win or draw

---

## 🧠 AI Logic

The computer:

* Selects a random empty cell
* Adds a slight delay (700ms) for better UX

---

## 📸 UI Highlights

* Dark background for modern look
* Cyan (X) vs Soft Red (O)
* Winning cells highlighted in green
